# 相对时间

发布时间如果距离当前时间 ＞24小时，则显示年月日；

24小时≥发布时间如果距离当前时间 ≥1小时，则显示XX小时前，

1小时＞发布时间如果距离当前时间 ，则显示XX分钟前，

 import moment from "moment";
 momentNowDate() {
      // 分
      let min = moment().diff(moment(data.create_time), "minute");
      // 小时
      let hour = moment().diff(moment(data.create_time), "hour");
      if (min <= 60) {
        return min + "分钟前";
      } else if (hour < 24) {
        return hour + "小时前";
      } else if (hour >= 24) {
        return moment(data.create_time).format("YYYY年MM月DD日");
      }
    },

TIP

moment().diff() //计算两个时间的差值

data.create_time //后台返回的创建时间
