# js

    function mrc() {
        var butong_net = new Array('1','2','3','4','5','6');  //创建数组
        var butong_net2 = Math.floor(Math.random() * butong_net.length); 
        var text = document.getElementById("text");
        text.firstChild.nodeValue = butong_net[butong_net2];
    }
        window.onload = mrc;

TIP

  • Math.random() * butong_net.length 随机选择数组(butong_net)长度以内的随机数
  • Math.floor() 向下取整(返回小于或等于一个给定数字的最大整数。)
  • Math.floor()
    Math.floor( 45.95); 
    // 45 
    Math.floor( 45.05); 
    // 45 
    Math.floor( 4 ); 
    // 4 
    Math.floor(-45.05); 
    // -46 
    Math.floor(-45.95); 
    // -46
